# UI Demonstration

This provides files that demonstrate the capabilities of the new UI.

## Contents
This includes examples for ELNs using the new layout system:

### Example ELN Models
- An ELN modeling a simple lab class about measuring electrical resistance.
- An ELN modeling a biography.

### Simulation Models
- Material properties and their analysis.
- Demonstrating DFT (Density Functional Theory) calculations, including bandstructure and Density of States analysis.
- Geometry optimization simulations.
